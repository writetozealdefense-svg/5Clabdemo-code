# Assets

This directory holds branding assets used by the PDF export scripts.

## Required: Zeal Defense Logo

Save the Zeal Defense logo here with this **exact filename**:

```
assets/zeal-defense-logo.png
```

- **Format**: PNG (transparent background preferred)
- **Recommended size**: 800x400px minimum
- **Used by**: `scripts/export-to-pdf.ps1`

If you don't have the PNG yet, save the image from the team chat/email as this exact path.

## Files

| File | Purpose |
|------|---------|
| `zeal-defense-logo.png` | Brand logo for PDF cover page (you must add this) |
| `pdf-style.css` | Stylesheet for PDF rendering (Zeal Defense colors) |
| `pdf-cover.html` | HTML template wrapper for cover page |
